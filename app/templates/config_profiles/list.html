About Configuration Profiles

Configuration profiles allow you to create reusable KAST scan configurations with custom settings for different scenarios:

  • Standard: Balanced settings for general-purpose scanning
  • Stealth: Low-profile scanning with reduced request rates
  • Aggressive: High-speed scanning for internal testing (power users only)
