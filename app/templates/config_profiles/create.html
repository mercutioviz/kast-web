Tips

  • Test your configuration with KAST CLI before adding to web
  • Create descriptive names and descriptions

YAML Configuration Help
Basic Structure

KAST configuration uses YAML format with two main sections:

  • global - Global settings that apply to all plugins
  • plugins - Plugin-specific settings
Available Plugins (Passive Scan)
  • mozilla_observatory - Mozilla Observatory scanning
  • subfinder - Subdomain enumeration
  • script_detection - JavaScript detection
  • wafw00f - WAF detection
  • katana - Web crawling
  • ftap - Technology profiling
Common Settings
  • timeout - Request timeout in seconds
  • rate_limit - Requests per second
  • concurrency - Number of concurrent workers
  • delay - Delay between requests in seconds
  • retry - Number of retry attempts
